# 智能简历生成器 - 前端使用说明

## 📋 项目简介

这是一个基于AI技术的智能简历生成器前端页面，使用原生HTML/CSS/JavaScript开发，无需构建工具，开箱即用。

## ✨ 功能特性

- 🎯 **智能生成**: 输入原始简历文本，AI自动生成结构化简历
- 🔗 **GitHub集成**: 自动分析GitHub仓库，提取项目信息
- 📂 **简历管理**: 加载、保存、删除简历数据
- 📥 **多格式导出**: 支持导出为PDF和JSON格式
- 📱 **响应式设计**: 完美支持PC、平板、手机访问
- 🎨 **现代化UI**: 简洁美观的渐变配色和动画效果

## 🚀 快速开始

### 1. 启动后端服务

确保后端Go服务正在运行（默认端口：8080）

```bash
# 设置必要的环境变量
export apiKey="your_api_key"
export GITHUB_TOKEN="your_github_token"  # 可选

# 启动服务
go run cmd/main.go
```

### 2. 访问前端页面

打开浏览器访问：`http://localhost:8080`

## 📖 使用指南

### 生成简历

1. **输入用户ID**: 填写您的用户标识（例如：user123）
2. **填写简历文本**: 在文本框中输入您的简历信息，格式示例：

```
姓名：张三
邮箱：zhangsan@example.com
电话：13812345678
地址：北京市朝阳区
职位：高级开发工程师

学历：
- 北京大学，计算机科学，本科，2015-2019

工作经历：
- 腾讯科技，高级开发工程师，2019-2023，负责微信后端开发
- 字节跳动，资深开发工程师，2023至今，负责抖音推荐系统

技能：Go、Python、Docker、Kubernetes、微服务、分布式系统

项目经验：
- 高并发消息系统：开发了日均处理10亿条消息的分布式消息系统
```

3. **点击生成**: 点击"生成简历"按钮，稍等片刻即可看到结构化的简历

### 添加GitHub项目

1. **输入GitHub链接**: 在GitHub仓库链接输入框中填写完整的仓库URL
   - 例如：`https://github.com/username/awesome-project`

2. **点击添加**: 点击"添加GitHub项目"按钮
   - AI会自动分析仓库的README和代码结构
   - 提取项目名称、技术栈、项目描述等信息
   - 自动添加到您的简历中

### 加载已有简历

1. **输入用户ID**: 填写要查询的用户ID
2. **点击加载**: 点击"加载已有简历"按钮
3. **查看简历**: 系统会从数据库中加载该用户的简历数据

### 导出简历

#### 导出为JSON
- 点击右上角"📥 JSON"按钮
- 下载包含完整简历数据的JSON文件
- 可用于数据备份或与其他系统集成

#### 导出为PDF
- 点击右上角"📥 PDF"按钮
- 浏览器打印对话框会自动打开
- 选择"另存为PDF"即可保存为PDF文件
- 页面已针对打印进行优化，确保输出美观

## 🎨 界面说明

### 左侧输入区
- **用户ID**: 必填，用于标识用户
- **原始简历文本**: 必填，输入您的简历原始信息
- **GitHub链接**: 可选，添加GitHub项目到简历
- **操作按钮**:
  - ✨ 生成简历
  - 🔗 添加GitHub项目
  - 📂 加载已有简历

### 右侧展示区
- **简历预览**: 实时展示生成的结构化简历
- **基本信息**: 姓名、邮箱、电话、地址、职位
- **教育背景**: 学校、专业、学位、时间
- **工作经历**: 公司、职位、时间、描述、成就
- **项目经验**: 项目名称、角色、描述、技术栈、亮点
- **技能特长**: 技能标签云展示

## ⌨️ 快捷键

- `Ctrl + Enter`: 在简历文本框中快速生成简历
- `Enter`: 在GitHub链接框中快速添加项目
- `Ctrl + P`: 打印/导出PDF

## 🐛 常见问题

### 1. 点击按钮没有反应？
- 检查浏览器控制台是否有错误信息
- 确认后端服务是否正常运行
- 验证API地址配置是否正确

### 2. 生成简历失败？
- 确保API Key已正确配置
- 检查原始文本格式是否合理
- 查看后端日志获取详细错误信息

### 3. GitHub项目添加失败？
- 验证仓库URL格式是否正确
- 确认仓库是否为公开仓库
- 检查GitHub Token是否配置（私有仓库需要）

### 4. 样式显示异常？
- 清除浏览器缓存
- 确认CSS文件路径正确
- 检查浏览器是否支持现代CSS特性
